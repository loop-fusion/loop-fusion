# Terminology

To not confuse users, this document describes what terminology is used for what.

| Term          | Description                                                                               |
|---------------|-------------------------------------------------------------------------------------------|
| Loop          | A normal loop. Executes code `N` times.                                                   |
| Looper        | Class that contains the range and the functions that should be executed.                  |
| Looper Union  | A union of loopers with different ranges.                                                 |
| Looper Merge  | Process of merging two or more Looper so that overlapping ranges are joined/merged/fused. |
| Looper Fusion | Alias for "Looper Merge".                                                                 |
